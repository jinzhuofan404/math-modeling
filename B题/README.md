# B题：策略类游戏的用户留存与付费策略优化

## 项目概述

基于游戏"代号K"的真实玩家行为数据（2000训练用户 + 1000测试用户，共~6M条记录），建立数学模型分析玩家留存规律、资源-付费关联，并设计新服首月付费策略。

## 使用的数学模型

| 问题 | 模型/方法 |
|------|----------|
| 问题1 | Kaplan-Meier生存估计、Nelson-Aalen风险函数、Cox比例风险模型 |
| 问题2 | Pearson相关性分析、Logistic回归、XGBoost回归 + SHAP可解释性 |
| 问题3 | K-Means聚类、多目标优化、蒙特卡洛模拟 |
| 问题4 | 定性分析（社交行为数据 + 实时战力数据建议） |

## 环境要求

- Python 3.x
- pandas, numpy, scipy, scikit-learn
- lifelines（生存分析）
- xgboost, shap（特征重要性）
- matplotlib（可视化）

```bash
pip install pandas numpy scipy scikit-learn lifelines xgboost shap matplotlib
```

## 运行方法

按顺序执行（特征工程必须先运行）：

```bash
# Step 0: 特征工程（生成 data/player_features.csv）
python 特征工程.py

# Step 1-4: 各问题求解（可独立运行）
python 问题1_求解.py
python 问题2_求解.py
python 问题3_求解.py
python 问题4_求解.py
```
