\section{玩家留存规律与流失预测模型}

\subsection{数据预处理与特征工程}

原始数据由两类文件组成：pickdata1和pickdata2。由于数据量庞大，采用分块读取策略进行处理。

\textbf{特征窗口约束。}问题1要求基于前3天行为预测30天留存，因此严格限定仅使用第1--3天的数据。Day1定义为玩家注册日期（register\_time），若缺失则以首次pickdata2记录日期替代。提取的14个前3天特征包括：登录天数（days\_logged\_d3）、第3天等级（level\_d3）、等级变化（level\_change\_d3）、日均在线时长（avg\_duration\_d3）、各类资源消耗量（food/wood/stone/diamond/coins\_reduce\_d3）、第3天钻石余额（diamond\_d3）与金币余额（gold\_d3）、是否首充（is\_pay\_d3，由first\_pay\_time $\leq$ Day3判定）、是否入盟（is\_league\_d3）及事件类型数（n\_event\_types\_d3）。\textbf{绝对排除}以下全周期变量以避免未来信息泄漏：lifecycle\_days、level\_end、total\_pay、vip\_level\_max、total\_records、diamond\_median。

\subsection{Kaplan-Meier留存率分析}

\subsubsection{模型建立}

Kaplan-Meier（KM）估计量\cite{kaplan1958}是一种非参数方法，用于估计生存函数$S(t)$——即玩家在注册后第$t$天仍然活跃的概率。为全篇统一口径，定义玩家$i$的流失指标基于活跃跨度（lifecycle\_days）：
\begin{equation}
T_i = \min(\text{lifecycle\_days}_i,\ 30),\quad \delta_i = \mathbb{I}(\text{lifecycle\_days}_i < 30)
\end{equation}
即若玩家首末活跃跨度不足30天则判定为流失事件（$\delta_i=1$），否则在第30天右删失（$\delta_i=0$）\cite{kleinbaum2012}。KM估计量为：

\begin{equation}\label{eq:km}
\hat{S}(t) = \prod_{i: t_i \leq t} \left(1 - \frac{d_i}{n_i}\right)
\end{equation}

其中$d_i$为$t_i$时刻流失的玩家数，$n_i$为$t_i$时刻仍在风险集中的玩家数。该估计量允许数据存在右删失。

\subsubsection{结果分析}

KM留存曲线如图\ref{fig:p1_km}所示。计算得到各级留存率如下：

\begin{itemize}
    \item 次日留存率（$R_1$）：\textbf{42.30\%}
    \item 7日留存率（$R_7$）：\textbf{22.95\%}
    \item 14日留存率（$R_{14}$）：\textbf{16.10\%}
    \item 30日留存率（$R_{30}$）：\textbf{7.05\%}
\end{itemize}

中位生存时间仅为\textbf{1天}，表明超过半数的玩家在注册后24小时内流失，该游戏面临严重的早期留存危机。30日留存率7.05\%与问题3基线留存率一致，全篇标签口径统一。

\subsection{流失高峰识别}

\subsubsection{模型建立}

为识别流失的关键时间节点，采用Nelson-Aalen估计量\cite{aalen1978}计算累积风险函数$H(t)$，进而通过差分得到风险函数$h(t)$的估计：

\begin{equation}
h(t) = \frac{\text{在}t\text{时刻流失的玩家数}}{\text{在}t\text{时刻仍处于风险集中的玩家数}}
\end{equation}

对原始风险函数进行3天滑动窗口平滑处理，识别$h(t)$的局部极大值点作为流失高峰。

\subsubsection{结果分析}

风险函数图见图\ref{fig:p1_hazard}。活跃跨度分布分析显示：

\begin{itemize}
    \item \textbf{第2天}是生命周期结束的最高峰，57.7\%的玩家活跃跨度仅为1--2天
    \item 第3天为次高峰，19.9\%的玩家活跃跨度止于2--3天
    \item 第4--5天活跃跨度终止率仍约9--10\%，此后衰减放缓
\end{itemize}

这一发现提示运营团队应将最核心的留存干预资源集中在注册后的前48小时内。

\subsection{分段留存对比分析}

为探究不同玩家群体的留存差异，将玩家按付费状态和联盟参与状态分别进行分层KM分析。结果如图\ref{fig:p1_seg}所示，对比极为显著：

\begin{itemize}
    \item \textbf{付费玩家} 7日留存率 \textbf{68.0\%} $\quad$ vs $\quad$ \textbf{非付费玩家} 7日留存率 \textbf{21.2\%}（差距3.2倍）
    \item \textbf{入盟玩家} 7日留存率 \textbf{61.5\%} $\quad$ vs $\quad$ \textbf{未入盟玩家} 7日留存率 \textbf{13.4\%}（差距4.6倍）
\end{itemize}

Log-rank检验结果表明两组差异均具有高度统计显著性（$p < 0.001$）。这一发现具有重要的策略含义：引导新玩家尽早完成首充并加入联盟，是提升整体留存率的最有效杠杆。

\subsection{Cox比例风险流失预测模型}

\subsubsection{模型建立}

为实现基于前3天行为数据的留存天数预测，采用Cox比例风险模型\cite{cox1972}。该模型是半参数模型，不要求指定基准风险函数的具体形式，而是将各协变量的影响建模为对基准风险的乘积效应：

\begin{equation}\label{eq:cox}
h(t \mid X) = h_0(t) \exp(\beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_k X_k)
\end{equation}

其中$h_0(t)$为基准风险函数，$X = (X_1, \ldots, X_k)$为协变量向量，$\beta_k$为回归系数。$\exp(\beta_k) > 1$表示该特征增加流失风险（``风险因子''），$\exp(\beta_k) < 1$表示该特征降低流失风险（``保护因子''）。

选取14个严格限定于前3天的特征：登录天数、第3天等级、等级变化、日均在线时长、各类资源消耗量、钻石/金币余额、是否首充、是否入盟及事件类型数。\textbf{绝对排除}全周期变量（lifecycle\_days、level\_end、total\_pay等），避免预测目标与特征共享未来信息。

\subsubsection{模型训练与评估}

将数据集按7:3分层划分为训练集和测试集。采用惩罚似然估计（penalizer=0.1）降低过拟合风险。模型评估指标：

\begin{itemize}
    \item 一致性指数（C-index）：\textbf{0.826}
    \item 预测平均绝对误差（MAE）：\textbf{4.36天}
\end{itemize}

\textbf{比例风险假设诊断与模型对照。}显示2个协变量存在时间效应递减（\texttt{days\_logged\_d3}, $p{=}0.0001$; \texttt{n\_event\_types\_d3}, $p{=}0.014$），其余12个满足PH假设。Schoenfeld残差图（图\ref{fig:p1_schoenfeld}）显示违背最严重的协变量残差随时间递减，说明其保护效应在早期最强。本文保留Cox模型——其C-index=0.826排序能力良好且系数解释性优于非参数替代。以随机生存森林（RSF）\cite{ishwaran2008}作对照：RSF的C-index=0.768，低于Cox。在有限样本下Cox的半参数结构具有偏差--方差权衡优势，本文以Cox为主模型、RSF为对照。

\subsubsection{特征重要性分析}

图\ref{fig:p1_cox}展示了各特征的回归系数及其95\%置信区间。主要发现：

\begin{itemize}
    \item \textbf{最强保护因子}（$\exp(\beta) < 1$）：
    \begin{itemize}
        \item 事件类型数（n\_event\_types\_d3）：$\beta = -0.43$，$p < 0.001$\\
        前3天内参与的游戏活动种类越多，流失风险越低——行为多样性是留存的最强信号
        \item 钻石余额（diamond\_d3）：$\beta = -0.18$，$p < 0.001$\\
        第3天钻石存量高的玩家投入更深，流失风险更低
        \item 登录天数（days\_logged\_d3）：$\beta = -0.17$，$p < 0.001$\\
        前3天每日登录的玩家具有显著更高的留存概率
    \end{itemize}
    \item \textbf{风险因子：}模型中唯一达到统计显著的风险因子为金币余额（gold\_d3，$\beta = +0.08$，$p = 0.01$），提示前3天金币存量偏高可能意味着玩家尚未找到有效的资源消耗出口，间接增加了流失风险。其余特征的风险效应均不显著，模型的流失预测主要依靠保护因子的强度。
\end{itemize}
